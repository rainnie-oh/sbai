# Changelog

All notable changes to the AI-PTI project will be documented in this file.

## [2026-04-12] - 16 人格图库上线与 PNG 插画同步

### 修改内容
- **插画系统升级**:
    - 完成了 `PersonaAvatar` 组件的重构，由原有的 SVG 手绘风格全面切换为用户提供的 PNG 精美插画。
    - 建立了自动映射机制，直接调用 `/public/illustrations/{type}.png` 路径。
- **全新“16 人格”图库页**:
    - 新增了 `GalleryView`，以阵营为单位汇总展示全部 16 种电子人格。
    - 采用了沉浸式卡片设计，展示人格形象、缩写、名称及经典金句。
- **人格档案详情页**:
    - 实现了静态人格详情页 `TypeDetailView`，支持从图库点击深入查看。
    - **特质语义化**: 将原本抽象的代码（如 FSCP）转化为直观的文字描述（如“消费观：克制”），方便用户理解。
    - 保留了“分析结果”页的视觉框架，但移除了针对个人的打分百分比，转而展示该人格的“满分”倾向。
- **Landing Page 精简**:
    - 移除了首页原本展示 16 人格的模块及底部的重复 CTA，使内容更加干练、直达核心。
- **交互与导航**:
    - 优化了全局导航栏，将“16 人格”按钮直接挂载至图库页。
    - 完善了返回机制，支持从“详情页”回退至“列表页”，从“列表页”回退至“首页”。

## [2026-04-12] - 引入相似人格推荐系统

### 修改内容
- **智能推荐模块**: 
    - 移除了底部全量 16 人格的网格展示，替换为“你可能还像...”推荐板块。
    - 开发了 `getSimilarPersonalities` 算法，基于阵营（Camp）一致性与维度（Dimension）相似度进行加权打分。
    - 目前推荐位展示 3 个分值最高的人格卡片，卡片包含头像、类型、名称及金句。
    - 该板块目前为静态展示（暂不支持点击跳转），匹配后续人格详情页开发。
- **UI 优化**: 
    - 为推荐卡片设计了磨砂玻璃质感与色彩呼吸感底效，符合整体 Premium 审美。
    - 调整了“重新测试”按钮的位置，布局更加平衡。

## [2026-04-12] - 修复评分逻辑 bug 与 16 种人格全映射

### 修复内容
- **修复全量“极致白嫖党”漏洞**: 
    - 修复了 `calculateQuizResult` 在处理答题数据时可能存在的类型匹配问题（兼容了数字与字符串 Key）。
    - 优化了 16 位维度的组合映射逻辑，确保 16 种可能性（2^4=16）都能准确对应到相应的人格原型。
    - 增加了 fallback 机制：当生成的 4 字母代码在映射表中缺失时，会尝试按维度特征进行模糊匹配，不再直接跳转至默认项。
- **数据同步**: 修正了 `scoring-rules.json` 中的拼写错误，将非标代码（如 `ADSV`, `FSDE`）修正为符合 4 轴定义的标准代码。
- **精细化次级区分**: 优化了 `resolveSecondary` 的逻辑，确保在 ASCP, ADCP 等重合组中，能根据特定关键题项准确分流。

## [2026-04-12] - AI-PTI 结果页优化与结构重构

- **文档补充**:
    - 新建 `16_personalities_info.md` 用于以表格形式管理 16 型人格的具体文案。
