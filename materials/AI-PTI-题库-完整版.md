## 评分机制

### 四个维度说明

**维度一：消费观 Token Economy**
- **F (克制收敛)**：精打细算、热衷白嫖、算力焦虑、能省则省
- **A (挥霍无度)**：算力自由、公费福报、上下文拉满、浪费算力无所谓

**维度二：控制欲 Control Desire**
- **D (强势驯化)**：PUA AI、严格要求、反复修改、掌控一切
- **S (随缘躺平)**：AI 说啥是啥、懒得教、随缘摆烂、佛系接受

**维度三：情感值 Emotional Bond**
- **C (冷漠工具)**：纯工具人、效率至上、无情绪、不客套
- **E (情感投射)**：谢谢辛苦了、深夜倾诉、AI 是唯一懂我的、情感寄托

**维度四：创作力 Creative Mode**
- **P (精密产出)**：JSON 格式、严谨逻辑、技术流、结构化需求
- **V (混沌意识流)**：抽象需求、意识流、整活、凭感觉

### 计分方法

**Step 1: 统计各维度得分**

每个维度单独计分，最后得到 4 个字母组合：

- **消费观**：F 分 > A 分 → F，否则 → A
- **控制欲**：D 分 > S 分 → D，否则 → S
- **情感值**：C 分 > E 分 → C，否则 → E
- **创作力**：P 分 > V 分 → P，否则 → V

**Step 2: 组合成四字母代码**

例如：
- F + D + C + P = **FDCP** (对应某个人格)
- A + S + E + V = **ASEV** (对应另一个人格)

## 完整算法伪代码

```javascript
// Step 1: 初始化各维度得分
let scores = {
  F: 0, A: 0,  // 消费观
  D: 0, S: 0,  // 控制欲
  C: 0, E: 0,  // 情感值
  P: 0, V: 0   // 创作力
};

// Step 2: 根据用户答案累加分数（参考上面的评分表）
// 例如：第1题选A → scores.F += 3

// Step 3: 确定四个维度的字母
let result = '';
result += (scores.F >= scores.A) ? 'F' : 'A';
result += (scores.D >= scores.S) ? 'D' : 'S';
result += (scores.C >= scores.E) ? 'C' : 'E';
result += (scores.P >= scores.V) ? 'P' : 'V';

// Step 4: 根据四字母组合 + 次级特征确定最终人格
let personality = '';

switch(result) {
  case 'FSCP':
    personality = 'POOR';
    break;
    
  case 'ASCP':
    // 次级区分逻辑
    if (answers[6] === 'C') {
      personality = 'SOLO';
    } else if (answers[2] === 'C' || answers[11] === 'C') {
      personality = 'LAZY';
    } else {
      personality = 'RICH';
    }
    break;
    
  case 'ADCP':
    if (answers[3] === 'A') {
      personality = 'SPEC';
    } else if (answers[2] === 'A' || answers[11] === 'A') {
      personality = 'PUAA';
    } else {
      personality = 'ANTI';
    }
    break;
    
  case 'ADSV':
    personality = 'LUCK';
    break;
    
  case 'FSDE':
    personality = 'PLEAS';
    break;
    
  case 'ASDE':
    if (answers[18] === 'A') {
      personality = 'MING';
    } else if (answers[7] === 'C' && answers[12] === 'C') {
      personality = 'LOVE';
    } else if (answers[18] === 'C') {
      personality = 'BABY';
    } else {
      personality = 'LOVE';
    }
    break;
    
  case 'FSDV':
    if (answers[9] === 'C' || answers[16] === 'C') {
      personality = 'MEME';
    } else {
      personality = 'MUSE';
    }
    break;
    
  case 'FSDP':
    if (answers[13] === 'A') {
      personality = 'DIST';
    } else {
      personality = 'PROMPT';
    }
    break;
    
  // ... 其他组合
}

return personality;
```
